Hi!

A little about me. About 3 years ago I began my coding journey. I dedicated a year to develop the basic HTML, CSS and Javascript skills so that I could create my first portfolio and find my first job as a developer, which I have been doing for the past 2 years.

I am a self-taught developer, and so I consider myself a lifelong learner who always keeps up to date with the current web development trends without going mad in the process. Jokes aside, I honestly love this industry and I think this is an exciting time, especially for Javascript enthusiasts.

Right now I would consider myself a mid-level developer in the HTML (HTML5 and mail templates), CSS (CSS3, Bootstrap 3 and 4, CSS Grid, Flexbox, Sass), Javascript (ES5, ES6, jQuery) and the Wordpress world (custom themes, theme development, e-commerce, any kind of tweak or addition to existing themes).

For the past few months I have started to pick up on some new technologies such as React or Node and all of the associated tools that come along, like npm and Webpack or Parcel. I am also interested in the JAMstack, the netlify CMS and to implement a headless CMS. I am currently playing with all that.

If you would like to contact me, in regards of anything, you can send me an e-mail here: ocasergi@gmail.com, or use the form in the About page, I'd be glad to hear from you.
